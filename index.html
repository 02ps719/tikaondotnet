Tika On DotNet

This project is a wrapper around the wonderful Tika library written in Java making it usable by .Net applications.

This may sound scary and heretical but did you know it is possible to leverage Java libraries from .Net applications with no TCP sockets or web services getting caught in the crossfire? IKVM baby!

For more details checkout this blog post from @KevM

Cloning

$ git clone git@github.com:KevM/tikaondotnet.git
$ cd tikadotnet
